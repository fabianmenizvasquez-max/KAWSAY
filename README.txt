KAWSAY | Landing Page Premium

Cómo abrir:
1. Descomprime la carpeta.
2. Abre index.html en Chrome, Edge o Firefox.

Archivos:
- index.html: estructura de la landing.
- style.css: diseño responsive y animaciones.
- script.js: menú móvil, animaciones, productos y FAQ.
- assets/images: logo y envase.

Pendiente para personalizar:
- Cambiar el número de WhatsApp en index.html.
- Cambiar el enlace de Instagram.
